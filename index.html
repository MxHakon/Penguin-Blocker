Penguin Blocker - Ad Blocker ("we", "our", or "us") is a comprehensive Chrome extension that provides advanced ad blocking, tracker prevention, and privacy protection. üéØ How We Use Information

All data collected by Penguin Blocker - Ad Blocker is stored locally on your device using Chrome's secure storage API and is used exclusively to:

  • Maintain Extension Functionality: Remember your blocking preferences and settings across browser sessions
  • Display Statistics: Show blocking counters and performance metrics within the extension popup
  • Domain Management: Remember which websites you've whitelisted or disabled blocking for
  • Custom Blocking: Apply your custom blocking rules and advanced filtering preferences
  • Performance Optimization: Cache settings for faster extension loading and operation
  • Data Export: Allow you to backup and restore your configuration when requested
